基于一致性-斥力耦合的多无人机圆形巡逻与队形保持仿真系统
项目简介
本项目实现了多无人机在二维空间内的圆形巡逻与队形保持的仿真平台。采用一致性-斥力耦合控制算法，支持障碍物避障、参数灵敏度测试及仿真数据可视化。
